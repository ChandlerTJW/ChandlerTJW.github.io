黑白腿舞蹈动作

发表于 2018-01-23 | | 阅读次数:
前奏：

双腿微蹲

第一部分
  1. 内八向前走向后走（前4后4）
  2. 斜踢腿（踢腿/放下/踢腿/放下, 左右各一次）
  3. 斜高抬踢腿（抬腿/踢腿/抬腿/踢腿，左右各一次）
  4. 斜高抬腿（抬腿x4，左右各一次）
第二部分
  1. 内八向前走向后走（前4后4）x 2
  2. 前后踢腿（前踢/后踢/前踢/后踢，左右各一次，后踢幅度稍大）
  3. 摆腿（抬腿/反向摆/摆回来/脚放下，左右各一次）
第三部分
  1. 内八向前走向后走（前4后4）x 2
  2. 单腿微侧向前跳（后腿跳前腿踢，前4后4）x 2
  3. 斜前踢，斜后踢（前踢/后踢/前踢/后踢 x 2, 左右各一次）
  4. 斜高抬腿（抬腿 x 8，左右各一次）
  5. 搭二郎腿
